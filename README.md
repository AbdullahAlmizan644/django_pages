# django_pages

<p>Simple 2 pages web app</p>

<h2>How to use:</h2>
git clone https://github.com/AbdullahAlmizan644/django_pages.git
</br>
cd django_pages
</br>
pip install pipenv
</br>
pipenv shell
</br>
pipenv install -r requirement.txt
</br>
python manage.py runserver
</br>
your app will be running on 127.0.0.1:8000
